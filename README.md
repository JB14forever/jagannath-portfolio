## 🛠️ Tech Stack & Key Features
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6)
- **Typography**: Inter (Google Fonts)
- **Styling**: Sleek modern design, custom variables, fully responsive layouts, hover animations, cards, and transitions.
- **Interactions**: Smooth scrolling, active navigation item highlighting, sticky header, mobile hamburger menu, interactive dark/light mode toggle.
- **Hosting**: Deployed on GitHub Pages.

## 🧑‍💻 Sections
1. **Hero Section**: Highlighting role search, Dublin location, resume download, and quick links.
2. **About Me**: Professional summary and 3 key metrics cards.
3. **Skills**: Grouped grid-based categories (Programming, BI, Machine Learning, Tools, etc.)
4. **Featured Projects**:
   - Automated Data Analytics Pipeline using AI Agents
   - Customer Churn Prediction using Machine Learning
   - Power BI Sales Performance Dashboard
   - Impact of Mobility Restrictions on Air Quality
5. **Experience**: Structured professional career timeline.
6. **Education**: Master's and Bachelor's degrees with highlights.
7. **Contact**: Quick-access email, LinkedIn, and GitHub links.
